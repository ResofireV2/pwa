# resofire/pwa

Progressive Web App extension for Flarum 2.x.

Adds install prompts, push notifications, offline support, and a full PWA manifest to your Flarum forum.

## Requirements

- Flarum `^2.0`
- PHP `^8.3`
- `ext-gmp` recommended (improves VAPID signing performance)

## Installation

```bash
composer require resofire/pwa
```

## Development

```bash
cd js && npm install && npm run dev
```
